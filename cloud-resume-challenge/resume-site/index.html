  • Cloud Resume Challenge

    The website you're looking at

    In the Cloud Resume Challenge, I successfully built and deployed a scalable and secure web application on AWS using a range of cloud and DevOps technologies. Leveraging Amazon S3 for static file storage, CloudFront for global content delivery, and AWS Certificate Manager for HTTPS security, the project showcased high availability and performance. AWS Lambda, API Gateway, and DynamoDB were employed to handle serverless computation, API management, and NoSQL database storage. Infrastructure provisioning was automated using CloudFormation, and the website was crafted with HTML, CSS, and JavaScript. Python scripts executed by Lambda were used to retrieve and store site visitor counts. The entire project was version-controlled on GitHub, with automation powered by GitHub Actions, providing a comprehensive demonstration of cloud and DevOps skills.
